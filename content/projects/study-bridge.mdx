Study Bridge is a web application designed to help students connect and collaborate on their studies. This project is built using React, Ant Design, and Tailwind CSS.

## Features

**User Authentication**: Users can sign up, log in, and log out of the application.  
**Dashboard**: A personalized dashboard that displays the user's profile information and study groups.  
**Study Groups**: Users can create, join, and manage study groups to collaborate with other students.  
**Chat Feature**: A real-time chat feature within study groups to facilitate communication and collaboration.


## Technologies Used

**React**: A JavaScript library for building user interfaces.  
**Ant Design**: A design system for enterprise-level products.  
**Tailwind CSS**: A utility-first CSS framework for rapidly building custom designs.  
**React Router**: A routing library for React applications.  
**Jest**: A JavaScript testing framework.  
**Firebase**: A platform for building web and mobile applications.
