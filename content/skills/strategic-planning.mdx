Real-time strategy isn't only in RTS games. Planning a CoD squad push requires knowing the map, anticipating enemy positions, and coordinating timing — all before the first shot fires.

Minecraft's late-game Redstone engineering demanded blueprinting complex systems before placing a single block. That habit of mapping the full system before starting the implementation is something I bring to every software project.

Strategic planning means knowing which battles not to fight, too.
